# Analyst guide

Start with collection quality. A completed investigation can still have optional sources skipped; a partial investigation has at least one source error.

Technology fingerprints and correlations should be reviewed through their evidence references before attribution. Moderate confidence means a deterministic provider signature matched one passive observation. High confidence means a source explicitly reported the relationship or multiple observations support it.

Use graph category controls to remove noisy source or service nodes. Search narrows the rendered set, selection highlights direct neighbors, and the side panel exposes relationship types and node metadata.

Historical events are observations, not proof that the same condition remains current. Certificate expiration records and archive captures should be interpreted using their timestamps.

Raw Evidence is authoritative. Derived sections are navigation and analysis aids built from that evidence.
